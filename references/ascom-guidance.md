# AFCM Guidance

Use this reference for Air Force Commendation Medal (`AFCM`) citation work in this skill's current medal set.

## Control note

- This file integrates the user-provided `ascm.txt` with the current policy stack.
- Where the source conflicts internally on medal naming or opening-sentence citation references, follow `DAFMAN 36-2806` for the actual award-specific opening and closing sentence pattern.

## AFCM drafting rules

- Draft from explicit source facts only
- Do not make up achievements, dates, or scope
- Use `DAFMAN 36-2806` opening and closing sentence rules for `AFCM`
- Keep the citation at or under `1350` characters including spaces
- Use `Courier New` `11`
- Use left and right justification when formatting the final package
- Spell everything out in citation prose
- Do not use abbreviations or acronyms in the citation unless the governing format makes one unavoidable
- Do not use periods after rank abbreviations
- Do not include quotations, embellishments, or classified content
- Use `SSgt Peter Snuffy` as the default placeholder identity unless the user explicitly wants a different placeholder

## Source-derived body expectations

- Use a formal opening sentence tied to the member, role, organization, and inclusive dates
- Build a narrative body that expands performance-report content into readable prose
- Include at least 2 to 3 significant accomplishments when the facts support that density
- Quantify outcomes wherever possible
- Use strong action verbs
- Prefer fully spelled-out organization and operation names in the final citation

## Narrative progression from the source

Use these transition patterns when they improve flow or when the local reviewer expects them:

- `During this period, ...`
- `Additionally, ...`
- `Moreover, ...`
- `Furthermore, ...`
- `Finally, ...`

Do not force every citation to use all five transitions if the result becomes bloated or repetitive.

## Number handling

- Spell out numbers `1` through `10`
- Spell out common two-word numbers when they are not part of a mixed numeric series
- If any value in a related series is `10` or more, express all values in that series as figures

## Missing-information rule

- If a required fact is missing and you cannot derive it from the provided source, use `(insert your information here)`

## Review checklist

- correct award-specific opening sentence
- correct award-specific closing sentence
- exact character compliance
- factual narrative only
- at least 2 to 3 meaningful accomplishments when supported
- approved acronyms only
- spelling and grammar checked
- passive voice reduced where it weakens clarity
