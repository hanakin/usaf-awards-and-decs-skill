# Decoration Intake

Use this file for the first response in decoration work.

## Default intake

- Ask only for the full EPB or OPB statements covering the medal period
- Ask for them in categorized form by report year and ALQ section
- Use year-tagged report labels such as `EPB23` or `EPB 2023`
- Keep the ALQ labels as `E`, `L`, `M`, `I`, and `H`
- Do not expand those labels into prose names during intake

Use this exact format:

EPB23:
E:
- statement
L:
- statement
M:
- statement
I:
- statement
H:
- statement

EPB24:
E:
- statement
L:
- statement
M:
- statement
I:
- statement
H:
- statement

## Default restrictions

Do not ask for:

- award period
- rank or full name
- duty title or unit
- strongest accomplishments
- citation versus justification choice
- a starter citation shell
- a direct draft before review

If the user already identified the medal family, do not ask them to restate it unless they indicate a different service or a different medal family.

## Fallback intake

Use this only when EPB or OPB source is unavailable.

1. Accomplishment statement:
   Metrics or measurable result:
2. Accomplishment statement:
   Metrics or measurable result:
3. Accomplishment statement:
   Metrics or measurable result:

Do not fall back to asking for:

- rank
- name
- duty title
