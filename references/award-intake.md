# Award Intake

Use this file for award-package intake.

## Default intake

Ask only for the inputs needed for the award product the user wants.

Gather:

- whether the format is individual statements or paragraph format
- the required character, line, or statement-count limits
- the scoring categories or section headings if they exist
- the accomplishment content to populate the award
- any local template or scoring sheet

## Statement-format award intake

When the user wants statement-format award content, gather:

- the required number of statements
- the accomplishment facts for each statement
- the metric or result for each statement
- the exact character limit if one applies

## Paragraph-format award intake

When the user wants paragraph-format award content, gather:

- the required line count
- the accomplishment facts to include
- the strongest accomplishment that should lead
- the measurable results that should survive compression

## Default restrictions

Do not default to citation-style intake for awards.
Do not ask for decoration-specific header metadata unless the user explicitly asks for it.
