# Decoration Drafting

Use this file only after split review and ranking review are complete.

## Drafting prerequisites

- Source material has been reviewed
- Split review is complete
- Ranking review is complete
- The user approved the final item set and order

Do not draft the citation before those steps are done unless the user explicitly tells you to skip them.

## Citation structure

- opening sentence
- narrative body
- closing sentence

Present the citation as normal paragraph text, not inside a code block.

## Citation body order

For a five-accomplishment body, use this preferred rule when the facts support it:

1. line 2 should be the strongest mission-impact accomplishment
2. line 3 should be the second strongest mission-impact accomplishment
3. line 4 should begin the organizational-impact transition and should have the lowest level of impact among the final three lines
4. line 5 should continue the organizational-impact transition and should have a higher level of impact than line 4 but less than line 6
5. line 6 should be the highest organizational-impact line in the body

Treat this as the ideal structure, not an absolute requirement.
If the available accomplishments do not support this exact pattern, preserve the strongest and clearest citation instead of forcing the order.
After ranking review, the user's approved final order controls.

## Organizational-impact examples

Organizational impact may progress like:

- Air Force
- United States Cyber Command
- Department of War

Use that kind of upward progression only when the source facts support it.

## Formatting and validation

- Use `During this period`, `Additionally`, `Moreover`, `Furthermore`, and `Finally` when the required format or local reviewer expects them
- Treat `1350` characters as a hard ceiling when using the 7-line format
- Count the fully assembled paragraph exactly
- Use the real intro length when checking compliance
- Preserve facts; do not introduce new metrics or claims
